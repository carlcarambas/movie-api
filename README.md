## Description

A movie api

## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker compose
$ docker-compose up or APP_PORT={PORT} docker-compose up

# development
$ yarn start

# watch mode
$ yarn start:dev

```

## Documentation
You can access documentation at:
0.0.0.0:{PORT}/api-doc/

## Test

```bash
# unit tests
$ yarn test

```
