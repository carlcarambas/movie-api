## Description

A movie api

## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker compose
$ docker-compose up or APP_PORT={PORT} docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentation
You can access documentation at:
0.0.0.0:{PORT}/api-doc/

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
